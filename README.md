# rr-crimelab
Crime data work

## Instructions
Run command, then check `dumps` folder

### Agencies list
```console
$ node dump-agencies-list
```

### Reports by agency id

```console
$ node dump-agency-reports id1 [..id2 ...id3] [startDate] [endDate]
# Example
$ node dump-agency-reports.js 65 20160620 20160625
```
